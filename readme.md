# Conkeror extension mode - Haivl

This is a page mode of conkeror web browser and[Haivl.com](haivl.com) Haivl is a comedy website in
Vietnam which is similar to 9gag. It helps users interact with button on
navigation bar:
1. *view new feeds*
2. *view unread feeds*
3. *view voted feeds*
4. *view video feeds*
5. *view hot feeds*
